DDB Fields
------------

This is a library that converts database values into display formats required
by EDU designs.

Setup
-----------
 $ virtualenv --python=python2.7   virtual_env
 $ source virtual_env/bin/activate
 $ python setup.py develop

Run tests:
 $ nosetests
